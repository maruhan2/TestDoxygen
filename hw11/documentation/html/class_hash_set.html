Detailed Description

template<typename T>
class HashSet< T >

Hashset for an arbitrary class T.

This class includes a class Element that has data members that holds a pointer to a T type object called data_ and a boolean called used_ that shows if the Element has been inserted into the hash.

Remarks
Deletion is not supported.
